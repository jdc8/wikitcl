This is a collaboratively-edited area on the web, dedicated to the Tcl programming language and it's extensions, including the Tk graphical toolkit.

What is Tcl?

Tcl is a simple-to-learn yet very powerful language. Its syntax is described in just a dozen rules, but it has all the features needed to rapidly create useful programs in almost any field of application - on a wide variety of international platforms

What is Tk?

Tk is a graphical toolkit for Tcl. It allows you to develop graphical aplications that run on Windows, Linux, MacOS X and many other platforms.
